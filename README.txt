INSTALL INSTRUCTIONS:

1. Download the .zip file

2. Extract the .zip file where you would like to (this path will be used to set up 
in eclipse make note of this)

3. Create an eclipse project using the three java files

4. Before running, click on Run Configurations... and go to the Arguements tab.

5. Put the path where you extracted the original zip file in the program arguements box,
 followed with "9999" (ex. "C:\Users\Steven\Documents\CSEFinal 9999")

6. Click run, the server should start to run, check the console to make sure

7. Choose whichever client you wish to use, we used Chrome, and type in localhost:9999

8. From there, navigate the page(s) and you're good to go
